# NLG-RL

## Requirement
* Python 3.X
* PyTorch 0.2X or 0.4X
* Numba

## Training
* Vocabulary predictor (minimal usage with the default settings used in our paper)<br>
`python train_vocgen.py --train_source XX --train_target YY --dev_source ZZ --dev_target WW`
